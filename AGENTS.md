# pi-ephemeral

Pi extension for ephemeral (temporary, unsaved) conversations.

## Commands

- `/ephemeral` — Toggle ephemeral mode
- `/ephemeral on` — Enable ephemeral
- `/ephemeral off` — Disable ephemeral

## Flags

- `--ephemeral` — Start pi in ephemeral mode

## Development

```bash
# Test locally
pi -e ./src/index.ts

# Install globally
pi install /Users/mitchfultz/Projects/AI/pi-ephemeral
```
